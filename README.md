# SC-JEcho

A live DPS graphing tool for *Star Conflict*.

---

## Overview

SC-JEcho is a lightweight desktop GUI that monitors your *Star Conflict* combat logs in real time and visualizes damage output as a live graph.

Point it at your log file (or the entire log folder), and it will continuously track and display incoming data as you play.

---

## Features

* 📊 Real-time DPS graphing from combat logs
* 📝 Live display of combat logs and chat in a single window
* 🔤 Adjustable font sizes for readability
* 🚫 Filter system to hide specific words or players
* 🧩 Toggleable UI elements for a cleaner view

---

## Usage

1. Launch the application
2. Select your *Star Conflict* logging folder or combat log file
3. Start playing — the graph updates automatically

> 💡 Best used on a second monitor for constant visibility during gameplay.

---

## Controls

Interaction is handled through the in-game chat (yes, intentionally).

Type:

```
!help
```

for a list of available commands and options.

---

## Notes

* Designed primarily for personal use and convenience
* UI/UX is intentionally minimal and functional
* Requires *Star Conflict* logging to be enabled

---

## Why?

Because watching your DPS in real time is useful… and funny
and controlling a desktop app through in-game chat confuses the hell out of your squadmates.
