# SC-JEcho
This is a live dps graph for Star Conflict.

---
This is just a small GUI window that can be pointed at your Star Conflict Logging folder, at the combat log (or the folder in general), and serves as a live graph of all of the damage being dealt.
It can also display combat logs and chat at the same time in the same window.
It features the ability to turn them off, change font sizes, and ban certain words or people from showing up in the log window.
- 
This is only really good if you have a 2nd monitor to put it in.
--
In order to interact with it you will have to use the in-game chat (because i think it's funny and no other reason).
I recommend typing `!help` if you are lost.
